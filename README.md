# Wikipedia Regulation Scraper

This Node.js project is designed to scrape regulation data from Wikipedia pages. It helps automate the extraction of regulatory information for research, analysis, or integration into other systems.

## Features
- Scrapes regulation-related data from specified Wikipedia articles
- Outputs data in structured formats (JSON, CSV, etc.)
- Easy configuration for different regulation topics or countries
- Handles Wikipedia page structure and updates

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm (Node Package Manager)

### Installation
```bash
git clone https://github.com/cyrinoorg/wikipedia-regulation-scraper.git
cd wikipedia-regulation-scraper
npm install
```

### Usage
1. Configure the target Wikipedia pages in the config file.
2. Run the scraper:
```bash
node index.js
```
3. Find the output data in the `/output` directory.
